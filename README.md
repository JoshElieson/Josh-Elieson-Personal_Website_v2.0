# Josh Elieson — Portfolio v2

Personal portfolio with layout inspired by [Helena Fu's site](https://helena-f.github.io/) (cleanfolio-style: hero, project cards, skills pills, contact).

## Local preview

Open `index.html` in a browser, or run a static server:

```bash
npx --yes serve .
```

## Customize

- **Portrait:** replace `Josh_Elieson_Portrait.png`
- **Project thumbnails:** replace files in `images/projects/` (same filenames, or update `src` in `index.html`)
- **Resume:** replace `JoshElieson_Resume.pdf`
- **Placeholders:** search for `[Placeholder]` and `href="#"` in `index.html`
- **Skills / projects:** edit sections in `index.html`

## Deploy

Static files only — suitable for GitHub Pages, Vercel, or Netlify (root = project folder).
