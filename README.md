# Josh Elieson — Personal Portfolio

My personal software engineering portfolio website, built to showcase my projects, experience, technical skills, and background as a Computer Engineering graduate and incoming Columbia MSCS student.

The site is designed to give recruiters and collaborators a quick, polished overview of who I am, what I have built, and where my technical interests are focused.

## Live Site

[joshelieson.com](https://joshelieson.com)

## Why I Built It

I built this portfolio to create a central place for my work, resume, GitHub, LinkedIn, and technical projects. The goal was to make my background easy to understand quickly while presenting my projects with a more polished and professional product feel.

## Features

- Responsive personal portfolio
- Project showcase section
- Experience and education overview
- Resume link
- Contact section
- GitHub and LinkedIn links
- Smooth animations and modern UI
- Mobile-friendly layout

## Tech Stack

- **Frontend:** React, TypeScript, Vite
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Deployment:** Vercel
- **Domain:** Porkbun DNS + custom domain
